# PHP Binary Download

## Windows Operating System:
Download it from [here](https://www.php.net/)

## For Linux User:
```shell
    apt-get install php
```
## CentOS
```shell
    yum install php
```
## For MAC User:
```shell
    brew install php
```

## PHP Installation Checking
```shell
    php
```
or
```shell
    php -V
```

* VS Code
